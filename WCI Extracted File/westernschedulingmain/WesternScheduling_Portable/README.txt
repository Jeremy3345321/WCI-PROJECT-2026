========================================
  WESTERN SCHEDULING SYSTEM
  Portable Package
========================================

PARA SA PRINCIPAL:

REQUIREMENTS:
  - XAMPP must be installed at C:\xampp
  - App files must be in C:\xampp\htdocs\westernschedulingmain\

PAANO GAMITIN:

STEP 1: SETUP (One-time)
  1. Install XAMPP sa C:\xampp
  2. Copy folder "app" to: C:\xampp\htdocs\westernschedulingmain\
  3. Import database_backup.sql via phpMyAdmin

STEP 2: GAMITIN (Every time)
  1. Double-click: START.bat
  2. Wait 10 seconds
  3. Browser opens automatically
  4. Use the system!
  5. Press any key to stop

========================================

ALTERNATIVE: Copy entire app folder to C:\xampp\htdocs\
Then access: http://localhost/westernschedulingmain/
